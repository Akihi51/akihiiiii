# 一级标题

段落之间使用空行分隔。你可以使用 _斜体_、**粗体** 和 `行内代码`。

列表写法如下：

- 第一项
- 第二项
- 第三项

> 这是引用块。
>
> 引用块可以包含多个段落。

## 二级标题

有序列表：

1. 第一步
2. 第二步
3. 第三步

代码块：

```ts
const message = "你好，Fuwari";
console.log(message);
```

表格：

| 名称 | 类型 | 说明 |
| --- | --- | --- |
| `title` | 字符串 | 文章标题 |
| `tags` | 数组 | 文章标签 |
| `draft` | 布尔值 | 是否为草稿 |

Markdown 很适合记录技术笔记、学习心得和日常想法。
